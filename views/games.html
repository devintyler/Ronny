Video Game Design

This page features video games that I have worked on in various roles including: Music Composer, Sound Designer, Creative Designer, and Audio Programmer

Caeludum

Current Project
Role: Music Composer
"Caeludum", is a turn-based god game in which players compete to become the most powerful deity by destroying or converting the followers of others. Choose what kind of god you want to be by unlocking new powers through out the game.

Chromacore

"Chromacore” is a 2D Musical Platforming Game where the player must build the game’s soundtrack by picking up items as they traverse the game’s level. Responsibilities included: designing level mechanics, appropriately editing songs, and extracting data from mp3 files using C# in the Unity Game Engine. Runner-up in GitHub’s “Game off” 2014

Enigmatic

"Enigmatic” is a 2D Platforming Game where the mechanics for each level vary based on a riddle that the player must answer correctly. Responsibilities included composing music for the game and creating all of the game’s sound effects. Runner-up in GitHub’s “Game off” 2014
